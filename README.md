# Numerical Methods in Chemical Engineering (NMCE) using Python

This repository contains Python programs for Numerical Methods taught in 3rd semester Chemical Engineering.

Topics covered:
- Bisection Method
- Newton Raphson Method
- Gauss Seidel Method
- Euler Method
- Runge Kutta Method

These codes are useful for solving engineering numerical problems using Python.
